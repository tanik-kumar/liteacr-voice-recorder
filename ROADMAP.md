# Roadmap

## Known issues

- microphone quality and loudness depend on the device hardware
- some OEM battery policies can interrupt background recording if the app is heavily restricted
- file browsing experience can vary between Android file manager apps

## Planned improvements

- cleaner app icon and release branding
- selectable audio quality presets
- recording rename support
- search and sort options in the recordings list
- optional waveform or recording timer improvements
