# Changelog

## [1.0.0] - 2026-03-22

### Added

- manual foreground audio recording with start, stop, pause, and resume
- recordings list with in-app playback, share, and delete actions
- shared storage output in `Music/LiteACR`
- release signing support through `keystore.properties`
- GitHub release notes and repo metadata docs
