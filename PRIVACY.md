# Privacy Policy

LiteACR is an on-device audio recorder.

## What LiteACR records

- microphone audio when you explicitly start a recording

## What LiteACR stores

- audio files saved in `Music/LiteACR`
- basic recording metadata exposed by Android for playback and file listing

## What LiteACR does not do

- it does not create an account
- it does not upload recordings to a remote server
- it does not sell or share recordings automatically
- it does not require cloud sync

## Permissions

- `RECORD_AUDIO` is used to record microphone audio
- `POST_NOTIFICATIONS` is used on Android 13+ to keep the foreground recording notification visible

## User control

- you decide when recording starts and stops
- you can play, share, and delete recordings from inside the app
- you can also manage files directly from the device storage location
